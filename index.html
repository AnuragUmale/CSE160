Assignment 0: Vector Library

View Site View Code

This project is a web-based application designed to visualize and interact with vector operations on a 2D canvas. Utilizing the cuon-matrix.js library, it facilitates the manipulation of vectors through operations such as addition, subtraction, multiplication, division, normalization, and magnitude calculation. Additionally, it offers features to compute angles between vectors and determine the area of a triangle formed by two vectors. The application aims to provide a hands-on learning experience for users interested in exploring vector mathematics. Users can input vector coordinates directly through a user-friendly interface, choose vectoroperations from dropdown menus, and see the results of these operations displayed both graphically on the canvas and numerically in the browserâ€™s console. Key functionalities include real-time graphical representation of vectors, interactive input fields for vector data, and immediate visual feedback on operations performed. This setup not only enhances understanding of vector concepts but also serves as a practical tool for anyone studying or working with mathematics or computer graphics. By offering an intuitive and interactive environment, the project makes complex vector operations accessible and engaging, ideal for educational purposes or as a resource for professionals in related fields.

Assignment 1: Painting

View Site View Code

The WebGL Paint Application assignment requires students to develop an interactive web-based painting program using WebGL. The application must allow users to draw geometric shapes (points, triangles, and circles) on a canvas through clicking and dragging operations, with customizable features controlled through UI elements including RGB color sliders, brush size adjustment, and circle segment count. The implementation needs to follow proper code organization with specific functions for WebGL setup, GLSL variable connections, and shape rendering, utilizing an object-oriented approach for shape management. The final component involves creating a hand-drawn reference picture using at least 20 triangles and implementing it programmatically using WebGL, while maintaining integration with the existing painting functionality. Students progress through 12 structured steps, starting from basic WebGL canvas setup and advancing to complex features like custom shape rendering and performance optimization.
