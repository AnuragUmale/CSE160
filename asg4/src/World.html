Notes to Grader

This application is developed from a previous project involving animated 3D models. Please note that the light settings (both point and spotlight) are exclusive; activating one will deactivate the other. Adjustments to lighting can be made using the sliders provided, but ensure that light animations are disabled beforehand. The RGB sliders allow for dynamic changes to light colors, enhancing the visual experience. For a more detailed demonstration of light interactions, activate the normal views and animations for the magenta and yellow cubes. Be cautious with the visualizer when using the spotlight, as it may overwhelm the display due to its extensive range.
